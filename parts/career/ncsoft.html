[AUDT_자동화 프로젝트] 오디오 후처리 자동화 프로그램 개발

기존 상업용 오디오 처리 프로그램이 원본 파형을 손상시키는 문제로 인해 팀원들이 수작업으로 데이터를 처리하고 있어, 높은 품질을 유지하면서 자동화하는 프로그램을 개발하였습니다.

  • 녹음실 미세소음 제거: 녹음된 성우의 음성 파일에서 녹음실의 미세한 잡음을 남김없이 제거
  • 숨소리 제거: F0 미검출 특성과 보정값을 이용하여 음성에서 숨소리를 제거
  • Endpoint Detection: TTS학습을 위해 맨 앞과 뒷 부분을 감지하여 공백을 일정하게 삽입
  • 텍스트 태깅: 대만어 데이터의 간체자 혼입을 태깅하여 대사 수정을 빠르게 할 수 있도록 구현
  • 문장 부호 공백 삽입: 오디오와 텍스트를 강제 정렬하여 각 문장 부호에 대한 공백 시간에 맞춰 공백을 삽입
  • 프로젝트 인원: 5명 · 역할: 개발, QA
  • 키워드: Text-to-Speech, Data Processing Automation, PyTorch, GUI, Numpy
