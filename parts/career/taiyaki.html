소속: Artificial Intelligence (계약직/재택)

[사내용 TTS 학습 도구 개발]

사내 사람들의 목소리로 TTS를 개발하려 했으나, 음성 데이터의 양이 적어 제대로 된 결과물을 기대할 수 없었고 파이프라인이 구축되어 있지 않았습니다.

  • 데이터셋 구축: 긴 음성 파일을 TTS 학습에 적합하게 분할하고 대사 파일을 자동 생성. 음성 변환 및 증강 기법으로 데이터 부족 문제 해결
  • 모델 학습: TTS 모델과 G2P 코드를 개발하고, 사전 학습 모델 기반으로 적은 데이터로도 미세 조정이 가능하도록 구현. 모델 압축을 통해 저장 효율성 향상
  • 성능 테스트: RTF로 추론 속도를, MOS로 발음 정확성을 측정. 하이퍼파라미터 조정으로 모델을 경량화하여 빠른 추론 속도 달성
  • 파이프라인 구축: TTS 모델을 실제 서비스에 배포하기 위한 추론 코드를 작성하여 파이프라인 구축
  • 프로젝트 인원: 2명 · 역할: 개발, QA
  • 키워드: Python, Text-to-Speech, Few-shot, Voice-Conversion, Deep-Learning
