supertonic.embed

SupertonicTTS의 비공개 Style Encoder 없이, 임의의 WAV에서 voice style JSON을 역추정하는 프로젝트입니다.
ONNX 모델을 PyTorch로 변환하여 gradient 역전파를 가능하게 하고, HuBERT 6개 레이어의 feature 통계 + Gram matrix를 perceptual loss로 사용하여 style 벡터를 최적화합니다.

  • 키워드: Python, Text-to-Speech, Gradient Optimization, HuBERT, ONNX-to-PyTorch, Voice Style Extraction

모델 최적화

음성 처리 모델을 경량화·최적화하여 on-device 환경에서 실시간 동작하도록 구현하였습니다.

  • DeepFilterNet3.c.wasm — 노이즈 제거 모델을 순수 C로 재작성. MacBook M2 ~1ms, Galaxy S23 ~4ms (최적화 리스트)

  • Project: LILAC — 학습 없이 3초 음성만으로 실시간 음성 변환. GPU 없이 동작

다국어 음성 합성 (VITS 기반)

VITS 아키텍처를 활용하여 다국어 음성 합성 연구 및 캐릭터 TTS 서비스를 개발하였습니다.

  • JA2ML-VITS — 일본어 데이터셋으로 19개국어 발화를 유도하는 다국어 TTS
  • JK-VITS — 한국어/일본어 이중 언어 TTS
  • RVC-VITS — RVC를 이용한 음성 데이터셋 변환 및 TTS 학습
  • ProsekaTTS — 프로젝트 세카이 캐릭터 TTS. 2026년 02월 기준 방문자 210만명 돌파
  • ShirokoTTS — 블루아카이브 스나오오카미 시로코 TTS 최초 개발
  • 키워드: Python, Text-to-Speech, Fine-Tuning, Phoneme-Conversion, Voice-Conversion, Data-Augmentation, Selenium, Distributed-Crawling
