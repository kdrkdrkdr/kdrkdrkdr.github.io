Extracting Voice Styles from Frozen TTS Models via Gradient-Based Inverse Optimization

Gyeongmin Kim. Preprint, 2026.04. To be submitted to ICASSP 2027 / Interspeech 2027.

TTS 모델의 비공개 Style Encoder에 접근하지 않고, 모델 가중치는 고정한 채 style conditioning 벡터만 최적화하여 임의의 WAV에서 voice style을 역추정하는 방법을 제안하였습니다. WavLM 기반 perceptual loss로 수렴시키며, 두 종류의 TTS 아키텍처(SupertonicTTS, Kokoro)에서 44명의 화자를 대상으로 검증하였습니다.

  • Preprint (DOI): 10.5281/zenodo.19646514
  • Code: supertonic.embed — SupertonicTTS 구현 (ONNX → PyTorch 변환으로 gradient 역전파 활성화)
  • Code: kokoro.embed — Kokoro TTS 구현 (preset voice 자동 선택 후 style 벡터 반복 최적화)
  • 키워드: Python, Text-to-Speech, Gradient Optimization, WavLM, ONNX-to-PyTorch, Voice Style Extraction, Inverse Optimization
